# 🎭 Model Agency Website

A full-stack model agency platform where users can submit their details and photos for modeling opportunities. The project includes contact and submission forms with image upload and email integration.

---

## 🚀 Features

- 📩 Contact Form (Email Integration)
- 📸 Model Submission Form with Image Upload (Front, Back, Left, Right)
- 📤 Email Notifications using Nodemailer
- 🖼️ Image Preview before upload
- 🧲 Drag & Drop Image Upload
- ❌ Remove Selected Images
- 📊 Upload Progress Bar
- 🔔 Toast Notifications (Success/Error)
- 🧾 Form Validation (No empty submission)

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Tailwind CSS
- React Hot Toast

### Backend:
- Node.js
- Express.js
- Nodemailer
- Multer
